🔧 EXPORT MAP + TEXTURES
① Export the .OBJ from UnrealEd
Open your map in UnrealEd 3 (UT2004 editor).
Go to File → Export → save as .obj.
This gives you the full map geometry with material names embedded.

② Export textures via the Texture Browser
In UnrealEd, open the Texture Browser (View → Texture Browser).
Load each texture package the map uses (File → Open in the browser).
Then use File → Export All to dump all textures as .bmp files.
Repeat for every package referenced by your map.

③ Or use UEViewer for better quality
Download UEViewer (umodel) — exports as .tga or .dds.
Open a .utx texture package → Tools → Export All.
The viewer supports .tga, .dds (DXT1/3/5), .bmp, and .png.

④ Import into this viewer
Use 📁 Import Folder and select the folder containing your .obj
plus all texture subfolders. The viewer matches material names to filenames automatically.

Tip: After import, check the Textures stat in the sidebar (e.g. 142/1075).
Missing textures = texture packages not yet exported. Open the console (F12) to see which filenames are missing.
